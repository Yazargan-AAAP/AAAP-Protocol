
# Security Policy

The security of the AAAP protocol is our top priority. We appreciate the community's help in identifying and reporting vulnerabilities so we can resolve them promptly.

## Reporting a Vulnerability

If you have discovered a potential security vulnerability, please **do not open a public issue.** Instead, report it privately to ensure we have time to address the issue before it is publicly disclosed.

Please send your report to: **[INSERT YOUR EMAIL ADDRESS HERE]**

### What to include in your report:
*   A clear description of the vulnerability.
*   Steps to reproduce the issue.
*   (Optional) Suggestions for potential fixes or mitigation strategies.

## Disclosure Policy
*   We will acknowledge receipt of your report as soon as possible.
*   We request that you keep the vulnerability confidential until we have had sufficient time to evaluate, patch, and release a fix.
*   Once a fix is deployed, we will credit the reporter in the release notes if they wish to be acknowledged.

Thank you for helping us keep the AAAP protocol secure.
