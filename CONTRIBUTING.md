# AAAP Protokolüne Katkıda Bulunma Rehberi

AAAP protokolüne gösterdiğiniz ilgi için teşekkür ederiz. Açık kaynaklı bir proje olarak topluluktan gelen katkıları değerli buluyoruz.

## Nasıl Katkıda Bulunabilirsiniz?
- **Hata Bildirimi:** Lütfen "Issues" sekmesini kullanarak detaylı bir rapor oluşturun.
- **Özellik Önerileri:** Protokol üzerindeki geliştirmeleriniz için öncelikle bir tartışma başlatın.
- **Kod Katkıları:** 
  1. Bu depoyu fork edin.
  2. Yeni bir branch oluşturun (`git checkout -b ozellik/yeni-katki`).
  3. Değişikliklerinizi yapın ve test edin.
  4. Değişikliklerinizi commit edin.
  5. Bir Pull Request gönderin.

## Kod Standartları
- Yazılan kodların temiz, dökümante edilmiş ve mevcut testleri geçiyor olması beklenir.
